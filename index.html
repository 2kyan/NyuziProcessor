Nyuzi

This project is a multicore, SIMD processor designed for highly parallel computation. It was inspired by Intel's Larrabee, although the instruction set and microarchitecture are substantially different. The design philosophy is to distribute work across many cache coherent cores, each with multiple hardware threads and a wide vector arithmetic unit.
